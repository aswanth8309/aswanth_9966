# Statistics and Trends

Repository: aswanth_9966  
Dataset: Titanic-Dataset.csv  

## Overview

This project analyses the Titanic dataset using Python. The script demonstrates data exploration, cleaning, statistical analysis, and visualisation techniques as required for the assignment.

## Files Included

- stats_trends_titanic.py  
- Titanic-Dataset.csv  

## How to Run

Make sure both files are in the same folder, then run:

python stats_trends_titanic.py

## What the Script Does

1. Loads and explores the dataset.
2. Cleans the data (removes duplicates, handles missing values, encodes categorical variables).
3. Calculates statistical moments (mean, variance, skewness, kurtosis) for the Fare variable.
4. Produces three plots:
   - Relational plot (Age vs Fare)
   - Categorical plot (Survival by gender)
   - Statistical plot (Correlation heatmap)

## Output

- Printed statistical summaries
- Interpretation of statistical moments
- Three labelled visualisations
